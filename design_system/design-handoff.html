04 — Patterns

Design Rules

01No rounded corners on cards or panels. Use rounded-none. Exceptions: nav dropdown rounded-lg, circular elements rounded-full.
02One red, one dark. Don't introduce new brand colors. The only accent is --primary red.
03container-portfolio always. Never build a section without wrapping content in .container-portfolio.
04Font roles are strict. font-heading (Space Grotesk) for headings. font-body (Inter) for body/meta. font-sanchari only for the సంచారి? wordmark.
05Background is global. The gradient+noise lives in body. Never recreate a textured background per-section. Use semi-transparent panel bg (bg-card/40) to let it breathe.
06Motion with intent. Animations signal importance — hero, interactive states, key transitions. Don't stack too many on one element.
07public/projects.json is the data source. Featured projects on Work page (featured: true). Non-featured on Fun page (featured: false).
